    SoundboardWeb.Layouts (soundboard v1.7.0)

    This module holds different layouts used by your application.

    See the layouts directory for all templates available. The "root" layout is a skeleton rendered as part of the application router. The "app" layout is set as the default layout on both use SoundboardWeb, :controller and use SoundboardWeb, :live_view.
